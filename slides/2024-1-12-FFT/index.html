What's the FFT?

FFT(Fast Fourier Transform), an algorithm invented by Fourier, always be used to calculate polynomial multiplication with large data range.

In this days, the main principles of FFT, such as Butterfly Transformation and Polynomial to Point Value, make a great difference in Compute Science and Network data processing.

How FFT works?

Recalling how you calculated polynomial multiplication in middle school

Must ne like this!

We calculate all the terms one by one!

Now we have $2$ polynomial $F$ and $G$

$F(x)=f_0x^0+f_1x^1+f_2x^2+\cdots+f_nx^n$

$G(x)=g_0x^0+g_1x^1+g_2x^2+\cdots+g_nx^n$

So it is:

$F(x)=\sum_{i=0}^nf_ix^i$

$G(x)=\sum_{i=0}^ng_ix^i$

Succeeding the previous idea, we get:

$(F\times G)(x) = \left(\sum_{i=0}^nf_ix^i\right)\cdot\left(\sum_{i=0}^ng_ix^i\right)$

So ugly, right?Consider contributions!

$(F\times G)(x)=\sum_{i=0}^n\sum_{k}f_ig_{i-k}x^i$

Get the $i$th term:

$[x^k](F\times G)=\sum_{k}f_ig_{i-k}x^i$

We will refer to the above equation as $convolution$.

$F*G = \sum\limits_{i+j=k}F_iG_j$

We've got a beautiful formula.But it is still slow to calculate.The time complexity is $O(n^2)$.In fact, FFT can do this in $O(n\log n)$

We should chage our mind!

Simple simplification formulas cannot solve this problem. So consider using point values to represent polynomials.
